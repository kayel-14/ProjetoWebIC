Web Scrapper para valores de ações

   Esse programa foi feito para entrar em sites com informações de valores de ações específicas, assim conseguindo armazenar seus valores e calcular seu patrimônio baseado neles.


                                                                                  

A modificação do código para incluir mais ações pode ser feita com facilidade, apenas incluindo o endereço do site(o código foi feito baseado no infomoney) e suas cordenadas na tela de display.

O código pode ser baixado aqui alt

Você também pode visualiza-lo abaixo.

import bs4
import time
from _datetime import datetime
import requests
import pygame
from bs4 import BeautifulSoup as Soup


def Sprice(url):
r = requests.get(url)
soup = bs4.BeautifulSoup(r.text, "html.parser")
p = soup.find_all('div', {'class': 'value'})[0].text
A = p[1:6].replace(',', '.')
A = A.replace('Re', '')
price = float(A)
print(price)
return price
